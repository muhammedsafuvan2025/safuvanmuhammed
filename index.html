01 — The Problem

Animals hiding in plain sight

Agricultural fields in Kerala face a serious issue — wild animals like boars and deer hide inside dense bushes at night, destroying crops. Traditional drones with RGB cameras can't see them. We built the solution for our final year engineering project at CUSAT.

02 — The Build

Built from scratch

We assembled the entire drone from raw components — carbon fibre frame, brushless motors, ESC controllers, a flight controller board, and a thermal imaging sensor module. Every part calibrated by hand. This wasn't a kit — we built it.

03 — The Tech

Seeing heat through foliage

The thermal sensor detects infrared radiation — animal body heat shows up as bright signatures even through dense bush cover. We mounted it on a gimbal for stable aerial capture. What's invisible to the human eye from 30 metres up becomes clearly visible on our feed.

04 — The Code

Python doing the thinking

We wrote Python scripts to process the raw thermal data in real time. The algorithm thresholds temperature ranges (animal body heat is 36–39°C), finds heat clusters in the frame, filters out environmental noise, and flags potential signatures with GPS coordinates for ground teams.
